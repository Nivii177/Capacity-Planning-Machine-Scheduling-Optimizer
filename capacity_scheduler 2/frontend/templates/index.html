About

SD Capacity Scheduler is a Python-powered web application that replicates the Excel-based capacity scheduling and solver functionality.

The backend uses SciPy's Linear Programming solver (HiGHS) to optimally allocate machine-days across products and machine types to maximize demand fulfillment. The algorithm minimizes unmet demand (shortfalls) subject to machine capacity constraints and product-machine compatibility rules.
